6. 适配器模式-（Adapter）
7. 模板方法模式-（Template）
8. 工厂模式-（Factory）
9. 抽象工厂模式-（Abstract Factory）
10. 职责链模式-（Chain of Responsibility）
11. 桥接模式-（Bridge）
12. 外观模式-（Facade）
